onitemclick 点击节点时的回调函数:onitemclick(node).
onloadbranch 加载延迟节点（isbranch=true 的节点）时的回调函数: onloadbranch(node,success,error).
- success(children) 延迟加载成功后的回调函数，参数为要添加到 node 下的子节点;
- error() 延迟加载失败后的回调函数.
onbeforeexpand 节点展开前的回调函数,返回 false 时阻止展开:onbeforeexpand(node).
onexpand 节点展开时的回调函数:onexpand(node).
onbeforecollapse 节点折叠前的回调函数,返回 false 时阻止折叠:onbeforecollapse(node).
oncollapse 节点折叠时的回调函数:oncollapse(node).
onbeforeselect 节点选中前的回调函数:onbeforeselect(node).
onselect 节点选中时的回调函数:onselect(node).
oncancelselect 取消节点选中时的回调函数:oncancelselect(node).
